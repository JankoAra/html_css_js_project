Osnovni atributi za elemente:
  • id - identifikuje jedan element, moze da se referencira taj element preko id u kodu
  • title - prikaz teksta kada se hover-uje preko komponente
  • class - povezuje element sa klasom za stilove u CSS, moze i lista klasa odvojenih razmakom
  • style - definise CSS stilove posebno za jedan element
Tagovi za grupisanje:
  • div - za grupisanje vise elemenata
  • span - za grupisanje vise inline elemenata (npr. da se vise reci u recenici formatira isto)
