Objasnjenja


Osnovni atributi za elemente:
  • id - identifikuje jedan element, moze da se referencira taj element preko id u kodu
  • title - prikaz teksta kada se hover-uje preko komponente
  • class - povezuje element sa klasom za stilove u CSS, moze i lista klasa odvojenih razmakom
  • style - definise CSS stilove posebno za jedan element
  • name - imenuje komponentu kako bi se adresirala u nekom linku (zastarelo - ne koristiti, koristiti id ili title umesto toga)
Tagovi za grupisanje:
  • div - za grupisanje vise elemenata
  • span - za grupisanje vise inline elemenata (npr. da se vise reci u recenici formatira isto)
Target atribut za linkove:
_blank
Otvara link u novom tabu
_self
Otvara link u istom tabu
_parent
Otvara link u parent frame-u
_top
Otvara link u top-level contekstu
customName
Otvara link u navedenom frame-u
Postavljanje boje linkova - atributi body tag-a
  • link - boja neposecenog linka
  • vlink - boja posecenog linka
  • alink - boja aktivnog linka (kada se pritisne)
Linkovi za download - samo referenciraju dokument (pdf, zip...)
